To draw a shape click on the shape tool then click once to start a shape and click again to end drawing the shape
To move a shape select it with the mouse and then click and and drag the shape to move it.
To save, load and create a new canvas selct the respective option from the file drop down
To change the colour click on the respective colour swatch or open a colour picker by clicking the picker button
To switch line thickness click on the line thickness that you'd like to use in the panel with the lines of varying thickness.
When you resize the window to larger it expands the canvas to fit the size of the window and if the window is resized smaller scrollbars will appear so the rest of the canvas is still accessible.
